# dotfiles 2021 by chezmoi

Digi-Angler's dotfiles managed by [chezmoi](https://www.chezmoi.io/).

## Get started

#### 1. Install chezmoi

```
brew install chezmoi
```

#### 2. Clone dotfiles

```
chezmoi init https://github.com/digiangler/dotfiles_2021.git
```

#### 3. Apply dotfiles

```
chezmoi apply
```
